## About The Project

[![Support screenshot][product-screenshot]](https://github.com/yuramorozov01/support)

This web-application provides API for a support system. 
* User can send ticket with question and send it.
* Support user can get open, closed and freezed tickets.
* Support user can send a message in all tickets.
* When support answer in ticket, ticket's author received email message (if email is specified during the registration).
* User can see messages in his tickets and answer on support messages.
* Support user can change ticket's status.
For more detailed information you can see API documentation, generated by Swagger (how to launch - see in the sections below).

## Getting Started

### Prerequisites

To run this program you have to follow this steps:

* Install [Docker Compose](https://docs.docker.com/compose/)
* Install [PostgreSQL](https://www.postgresql.org)
* Install (with pip) [Python](https://www.python.org/downloads/)

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/yuramorozov01/support.git
   git checkout dev
   cd support/
   ```
2. Create .env file in the folder src/ near manage.py
   ```
   SECRET_KEY=Secret key of your application
   DEBUG=True if want to turn on Debug mode (else - False)
   
   DB_ENGINE=Engine for database
   DB_NAME=Database name
   DB_USER=Database user
   DB_PASSWORD=Database user password
   DB_HOST=Database host
   DB_PORT=Database port

   EMAIL_USE_TLS=True if want to use TLS (else - False)
   EMAIL_HOST=SMTP host address 
   EMAIL_HOST_USER=Source email
   EMAIL_HOST_PASSWORD=Email password
   EMAIL_PORT=SMTP port (587 if EMAIL_USE_TLS is True)
   ```
3. Give execute root to run bash scripts
   ```sh
   chmod +x run.bash
   ```

<!-- Executing EXAMPLES -->
## Executing
1. Run Docker (launch Docker Desktop)
2. Run script
   ```sh
   ./run.bash
   ```
3. To create superuser: Open Docker Destop >> Find Containter / Apps tab >> Open CLI of task-support-* >> Type following command in CLI
   ```sh
   python3 ./support/src/manage.py createsuperuser
   ```
4. Open the following URL to see API documentation
   ```
   127.0.0.1:8000/swagger/
   ```
